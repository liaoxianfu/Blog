##运行环境
    python3.6
    Django==1.11.4
    django-ckeditor==5.4.0
    django-js-asset==1.0.0
    django-suit==0.2.25
    mysqlclient==1.3.12
    Pillow==5.0.0
    PyMySQL==0.8.0
    pytz==2018.3
安装的包在requirements.txt里，可以pip直接安装即可
